# LLM Review Scraper

Scraping Google Maps review using Puppeteer and create Sentiment Analysis summary using LLM

## Installation

Clone the repository and run npm install.

```bash
npm install
```

## Usage

```javascript

# change url '##newURL'
await page.goto("##newURL", { waitUntil: "domcontentloaded"});
```
